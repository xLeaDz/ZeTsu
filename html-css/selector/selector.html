Quel selecteur permet de:

  • Sélectionne TOUS les éléments de la page html: *
  • Sélectionne tous les éléments ayant le nom de class "ma-jolie-nav": .ma-jolie-nav
  • Sélectionne tous les éléments ayant le nom de mon id "mon-unique-id": #mon-unique-id
  • Sélectionne toutes les balises
    ,
    ET
    : div, article, aside
  • Sélectionne les liens inclus/imbriqués dans la balise
    : nav a
  • Sélectionne tous les liens de la nav et active une animation lors du survol de la souris: nav a:hover

Décrivez leur usage et leurs valeurs possibles

  1. background: couleur(ou image de fond). Valeur possible url, couleur(literal, hex, rgb, rgba) /!\ Le treme background est générique il contient en réalité toutes les propriétés background(background-image, background-color, background-size, background-repeat, ...)
  2. color: couleur de la police, valeur possible en litéral, hex, rgb, rgba, ...
  3. float: permet de faire flotté un élément, valeurs possible droite ou gauche
  4. font-family: défini la police d'un élément, valeur possible toutes polices préalablement installé
  5. font-size: taille de la police, valeurs possible px, em, rem, %, ...
  6. height: hauteur de l'élément, valeurs possible px, em, rem, %, ...
  7. left: position à partir de la gauche de la fenêtre, valeurs possible px, em, rem, %, ...
  8. line-height: hauteur de l'interligne, valeurs possible px, em, rem, %, ...
  9. margin: marge externe(ou exterieur), valeurs possible px, em, rem, %, ...
  10. padding: marge interne(ou intérieur), valeurs possible px, em, rem, %, ...
  11. position: position de l'élément dans la page(ou fenêtre), valeurs possible fixed(fixe), relative(relatif), absolute(absolu)
  12. text-align: alignement du texe, valeurs possibles left, center, right
  13. text-decoration: change les valeurs hérité par défaut de l'élément(pour un a par exemple ce sera l'underline)
  14. top: position à partir du haut de la fenêtre, valeurs possible px, em, rem, %, ...
  15. transition: temps de transition lors d'une animation
  16. width: largeur de l'élément, valeurs possible px, em, rem, %, ...

Propriétés CSS à rechercher

Recherchez les propriétés ci-dessous, faites des tests avec sur cette page depuis la console de débuggage pour tester les valeurs qu'elles peuvent prendre

  1. font-weight: défini le niveau de gras de la police, valeurs possibles normal, bold, 100, 500, 900, ...
  2. letter-spacing: espacement entre les lettre, valeurs possibles normal, px, rem, %, ch
  3. word-spacing: espacement entre les mots, valeurs possibles normal, px, rem, %, ch
  4. text-shadow: ajoute des ombres à texte, valeurs possibles px + hex, rgb, rgba, ...
  5. background-image: image de fond, valeur possible url
  6. background-size: taille de l'image de fond
  7. opacity: modifie la transparence de l'élément, valeur possible entre 0 et 1
  8. border-style: défini le style de la bordure, valeurs possibles dotted, solid, dashed, ...
  9. box-shadow: ajoute des ombres à la boîte via une liste d'ombres séparées par des virgules(px, em, %
  10. list-style-image: définit l'image utilisée comme puce devant les éléments de listes. On peut également utiliser la propriété raccourcie list-style.
  11. border: ajoute une bordure à l'élément
  12. border-radius: ajoute un arrondi à la bordure
  13. border-width: définit la largeur d'une bordure pour la boîte d'un élément(px, em, %
  14. font-variant: Détermine si l'élément doit être affiché tout en majuscules, les minuscules du code source étant affichées en majuscules plus petites. Si une famille de polices comprend une variante Petites capitales, le navigateur doit normalement utiliser cette dernière.
  15. background-color: couleur de fond(litéral, hex, rgb, ...)
  16. !important;: permet de spécifier que la propriété est la plus importante dans votre feuille CSS en lui donnant un poids plus important
